**Установка сервера:**

Чтобы запустить сервер, вам необходимо:

1. Установите все необходимые зависимости из файла requirements.txt (если существует).
2. Запустите скрипт `app.py` с помощью команды `python app.py`.
3. Сервер будет доступен по адресу `http://localhost:5000`.

**Создание продукта:**

Чтобы создать продукт, вы можете использовать следующую команду:

```
curl -X POST -H "Content-Type: application/json" -d '{"id": "новый_id", "name": "новое_имя", "price": 10.99}' http://localhost:5000/products
```

Замените `новый_id`, `новое_имя` и `10.99` на свои значения.

**Получение всех продуктов:**

Чтобы получить все продукты, вы можете использовать следующую команду:

```
curl -X GET http://localhost:5000/products
```

**Получение продукта по ID:**

Чтобы получить продукт по его ID, вы можете использовать следующую команду:

```
curl -X GET http://localhost:5000/product/<id>
```

Замените `<id>` на фактический ID продукта.
